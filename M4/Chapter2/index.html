ตำแหน่ง

ความเร็ว vs อัตราเร็ว

อัตราเร็วเฉลี่ย \(\neq\) ขนาดความเร็วเฉลี่ย ยกเว้นเคลื่อนที่แนวตรงโดยไม่กลับทิศ เพราะอัตราเร็วใช้ระยะทาง แต่ความเร็วใช้การกระจัด

ความเร่ง

ความเร่งเป็นลบ ไม่ได้หมายความว่าช้าลงเสมอไป ต้องดูทิศของความเร็วต้นด้วย: ถ้าทิศความเร็วและความเร่งตรงกัน = เร็วขึ้น; ตรงข้ามกัน = ช้าลง

การตกเสรี

วัตถุมวลมากและมวลน้อยที่ตกจากที่สูงเดียวกันจะตกถึงพื้นพร้อมกัน ถ้าไม่มีแรงต้านอากาศ (ทดลองโดยกาลิเลโอ) ทุกวัตถุมีความเร่ง \(g \approx 9.8\) m/s² เท่ากัน

2.1 ตำแหน่ง (Position)

ตำแหน่ง คือ จุดที่วัตถุอยู่ ณ เวลาใดเวลาหนึ่งเมื่อเทียบกับจุดอ้างอิง (reference point) สำหรับการเคลื่อนที่แนวตรง นิยมกำหนดแกน x ในแนวระดับ หรือแกน y ในแนวดิ่ง

ตำแหน่งเป็นปริมาณเวกเตอร์ ต้องระบุทั้งระยะห่างจากจุดอ้างอิงและทิศทาง เครื่องหมายบวก (+) หมายถึงด้านขวาของจุดอ้างอิง และลบ (−) หมายถึงด้านซ้าย

  • ต้องกำหนดจุดอ้างอิงก่อนเสมอ (เช่น ต้นไม้ เสาไฟ จุด O)
  • ตำแหน่งบนแกนเดียวกันอาจเป็นค่าบวก ศูนย์ หรือค่าลบ
  • ใช้สัญลักษณ์ \(x\) แทนเวกเตอร์ตำแหน่ง (ในแนวระดับ)
  • เขียนแบบย่อ: \(x(t=1\,\text{s}) = +8\,\text{m}\) หรือ \(x(1) = +8\,\text{m}\)
ตัวอย่าง: รถยนต์อยู่ห่างจากเสาไฟต้นที่หนึ่ง 4 เมตร ทางขวา → ตำแหน่ง \(x = +4\,\text{m}\) และคนอยู่ทางซ้าย 2 เมตร → \(x = -2\,\text{m}\)

2.2 การกระจัดและระยะทาง

ระยะทาง (distance) คือความยาวตลอดเส้นทางที่วัตถุเคลื่อนที่จริง เป็นปริมาณสเกลาร์ มีค่าเป็นบวกเสมอ หน่วย: เมตร (m)

การกระจัด (displacement) คือการเปลี่ยนตำแหน่งจากจุดเริ่มต้นไปจุดสุดท้าย เป็นปริมาณเวกเตอร์ อาจเป็นบวก ลบ หรือศูนย์ก็ได้

สมการการกระจัด
\[ \Delta x = x_f - x_i \]
คำอธิบายตัวแปร:
  • \(\Delta x\) = การกระจัด (เมตร, m)
  • \(x_f\) = ตำแหน่งสุดท้าย (เมตร, m)
  • \(x_i\) = ตำแหน่งเริ่มต้น (เมตร, m)
ความสัมพันธ์ระหว่างระยะทางและการกระจัด
\[ s \geq |\Delta x| \]
  • \(s\) = ระยะทาง (เมตร, m) — มีค่าไม่ติดลบ
  • \(|\Delta x|\) = ขนาดของการกระจัด (เมตร, m)
เมื่อวัตถุเคลื่อนที่กลับทิศ ระยะทาง > ขนาดการกระจัด
ตัวอย่าง 2.1 (จากตำรา): รถยนต์เริ่มจาก \(x(0) = +4\,\text{m}\) เคลื่อนที่ไป \(x(1) = +8\,\text{m}\) แล้วถอยกลับมา \(x(3) = -8\,\text{m}\)
ช่วง t = 0 ถึง t = 3 s:
การกระจัด = \((-8) - (+4) = -12\,\text{m}\), ระยะทาง = \(4 + 16 = 20\,\text{m}\)

เดินไปแล้วกลับ

ระยะทาง 10 m, การกระจัด 2 m

2.3 อัตราเร็วและความเร็ว

อัตราเร็วเฉลี่ย (Average Speed)

อัตราเร็วเฉลี่ยเป็นปริมาณสเกลาร์ ไม่มีทิศทาง บอกว่าวัตถุเคลื่อนที่เร็วแค่ไหน

\[ v_{av} = \frac{s}{\Delta t} \]
  • \(v_{av}\) = อัตราเร็วเฉลี่ย (m/s)
  • \(s\) = ระยะทางรวม (m)
  • \(\Delta t\) = ช่วงเวลา (s)

ความเร็วเฉลี่ย (Average Velocity)

ความเร็วเฉลี่ยเป็นปริมาณเวกเตอร์ มีทิศทาง บอกทั้งขนาดและทิศการเคลื่อนที่

\[ \bar{v}_x = \frac{\Delta x}{\Delta t} = \frac{x_f - x_i}{t_f - t_i} \]
  • \(\bar{v}_x\) = ความเร็วเฉลี่ยในแนวแกน x (m/s)
  • \(\Delta x\) = การกระจัด (m)
  • \(\Delta t\) = ช่วงเวลา (s)
  • \(x_f, x_i\) = ตำแหน่งสุดท้ายและเริ่มต้น (m)

ความเร็วขณะหนึ่ง (Instantaneous Velocity)

ความเร็วของวัตถุ ณ เวลาขณะหนึ่ง หาจากค่าความชันของเส้นสัมผัสกราฟ x-t ณ จุดนั้น (พิจารณาช่วงเวลา \(\Delta t \to 0\))

ข้อสังเกต: ในฟิสิกส์ "อัตราเร็ว" และ "ความเร็ว" เป็นคนละปริมาณ อัตราเร็วเฉลี่ย = ระยะทาง/เวลา (สเกลาร์), ความเร็วเฉลี่ย = การกระจัด/เวลา (เวกเตอร์)

การวัดความเร็วจากแถบกระดาษ (หน้า 62–66)

ในปฏิบัติการจริง เราใช้ เครื่องเคาะสัญญาณเวลา (Timer) ที่เคาะ 50 ครั้งต่อวินาที ทำให้ช่วงเวลาระหว่างจุดแต่ละจุดบนแถบกระดาษ = \(\frac{1}{50}\) s = 0.02 s

การหาขนาดความเร็วเฉลี่ยจากแถบกระดาษ (ตัวอย่าง 2.3)
\[ |\bar{v}_{x,AB}| = \frac{|\Delta x_{AB}|}{\Delta t_{AB}} \]
  • \(|\Delta x_{AB}|\) = ระยะทางระหว่างจุด A ถึง B วัดด้วยไม้บรรทัด (m)
  • \(\Delta t_{AB}\) = จำนวนช่วงจุดระหว่าง A ถึง B × 0.02 s
ตัวอย่าง 2.3 (หน้า 62): รถเคลื่อนที่จาก A ถึง B ระยะ 10.00 cm ใน 8 ช่วงจุด
\(\Delta t = \frac{8}{50}\) s  →  \(|\bar{v}_{x,AB}| = \frac{0.1000}{8/50} = 0.6250\) m/s = 62.50 cm/s

การหาขนาดความเร็วขณะหนึ่งจากแถบกระดาษ (หน้า 65–66)

เพื่อหาขนาดความเร็วที่จุด b ให้วัดระยะจากจุด a ถึงจุด c (จุดข้างหน้าและข้างหลัง b) ใน 2 ช่วงจุด แล้วคำนวณหาอัตราเร็วเฉลี่ยในช่วงนั้น ค่าที่ได้ถือเป็นขนาดความเร็วขณะหนึ่ง ณ จุดกึ่งกลาง (จุด b)

ความเร็วขณะหนึ่ง ณ จุด b (ตัวอย่าง 2.3, หน้า 65)
\[ v_b = \frac{|\Delta x_{ac}|}{\Delta t_{ac}} = \frac{|\Delta x_{ac}|}{2 \times 0.02\,\text{s}} \]
  • \(|\Delta x_{ac}|\) = ระยะทางระหว่างจุด a ถึง c (ข้ามจุด b)
  • \(\Delta t_{ac}\) = 2 ช่วงจุด = 2/50 s = 0.04 s
ตัวอย่าง (หน้า 65): จุด a ถึง c ระยะ 2.00 cm ใน 2 ช่วงจุด
\(v_b = \frac{0.0200}{2/50} = 0.500\) m/s = 50.0 cm/s
ตัวอย่าง 2.4 (หน้า 66): หาอัตราเร็วที่จุด A โดยใช้การกระจัดจากจุด O ถึง B (ใช้จุดก่อนหน้า O และหลัง B เพื่อครอบจุด A ไว้ตรงกลาง) ได้อัตราเร็วที่จุด A = 2.500 m/s

คำนวณเร็วเฉลี่ย

อัตราเร็วเฉลี่ย 2.50 m/s, ความเร็วเฉลี่ย 0.50 m/s

สังเกตว่าเดินวนกลับมาจุดเดิมได้: ระยะทางไม่เป็นศูนย์ แต่การกระจัดอาจเป็นศูนย์

2.4 ความเร่ง (Acceleration)

ความเร่ง คืออัตราการเปลี่ยนแปลงความเร็วต่อเวลา เนื่องจากความเร็วเป็นเวกเตอร์ การเปลี่ยนแปลงจึงเกิดได้ 3 แบบ:

  • ขนาดความเร็วเปลี่ยน แต่ทิศคงเดิม เช่น รถเร่งบนทางตรง
  • ทิศเปลี่ยน แต่ขนาดคงเดิม เช่น รถเลี้ยวโค้งด้วยอัตราเร็วคงตัว
  • ทั้งขนาดและทิศเปลี่ยนพร้อมกัน เช่น การแกว่งชิงช้า

ความเร่งเฉลี่ย (Average Acceleration)

\[ a_{x,av} = \frac{\Delta v_x}{\Delta t} = \frac{v_{fx} - v_{ix}}{t_f - t_i} \]
  • \(a_{x,av}\) = ความเร่งเฉลี่ยในแนวแกน x (m/s²)
  • \(\Delta v_x\) = การเปลี่ยนความเร็วในแนวแกน x (m/s)
  • \(v_{fx}\) = ความเร็วปลายในแนวแกน x (m/s)
  • \(v_{ix}\) = ความเร็วต้นในแนวแกน x (m/s)
  • \(\Delta t = t_f - t_i\) = ช่วงเวลา (s)
ข้อสำคัญ: ในฟิสิกส์ วัตถุที่เคลื่อนที่ช้าลงก็มีความเร่ง (เรียกว่า "มีความเร่งในทิศตรงข้ามกับความเร็ว") ความเร่งเป็นลบไม่ได้หมายถึงช้าลงเสมอ ต้องพิจารณาร่วมกับทิศของความเร็ว:
  • ความเร็วและความเร่ง ทิศเดียวกัน → วัตถุเร็วขึ้น
  • ความเร็วและความเร่ง ทิศตรงข้าม → วัตถุช้าลง

ความเร่งเฉลี่ย

a = 1.50 m/s²

ลองใส่ vf น้อยกว่า vi เพื่อดูความเร่งติดลบ (วัตถุช้าลง)

2.5.1 กราฟระหว่างตำแหน่งกับเวลา (x-t Graph)

\(\bar{v}_x = \dfrac{\Delta x}{\Delta t} = \dfrac{x_2 - x_1}{t_2 - t_1}\)

✏️

ลากเส้นสัมผัสกราฟที่จุดที่ต้องการ → วัดความชันของเส้นนั้น

📏
กราฟเส้นตรง = ความเร็วคงตัว (a = 0)

ความชันคงตัวหมายถึงความเร็วไม่เปลี่ยนตลอดการเคลื่อนที่

〰️
กราฟเส้นโค้ง = มีความเร่ง

ความชันเปลี่ยนไปตามเวลา แสดงว่ามีความเร่งเกิดขึ้น

ตัวอย่างจากตำรา (หน้า 65, รูป 2.13): รถยนต์มีตำแหน่ง x(0) = +4.0 m, x(1.0) = +8.0 m, x(2.0) = 0 m, x(3.0) = −8.0 m
  • ช่วง t = 0 ถึง 1.0 s: \(\bar{v}_x = \frac{(+8.0)-(+4.0)}{1.0} = +4.0\) m/s (เคลื่อนทางบวก)
  • ช่วง t = 1.0 ถึง 2.0 s: \(\bar{v}_x = \frac{0-(+8.0)}{1.0} = -8.0\) m/s (เคลื่อนทางลบ)
  • ช่วง t = 2.0 ถึง 3.0 s: \(\bar{v}_x = \frac{(-8.0)-0}{1.0} = -8.0\) m/s (เคลื่อนทางลบ)
สรุปสิ่งที่อ่านได้จากกราฟ x-t
ลักษณะกราฟ ความหมาย
เส้นตรงเอียงขึ้น (slope > 0) เคลื่อนที่ทิศบวก ความเร็วคงตัว
เส้นตรงเอียงลง (slope < 0) เคลื่อนที่ทิศลบ ความเร็วคงตัว
เส้นขนานแกนเวลา (slope = 0) วัตถุอยู่นิ่ง
เส้นโค้งขึ้น (cong up) ความเร็วเพิ่มขึ้น (เร่ง)
เส้นโค้งลง (cong down) ความเร็วลดลง (หน่วง)
จุดเปลี่ยนทิศ (ตำแหน่งสูงสุด) ความเร็วขณะหนึ่ง = 0

กราฟ x-t: แสดงตำแหน่งของรถยนต์ตามเวลา ความชันบ่งบอกความเร็ว

v-t

2.5.2 กราฟระหว่างความเร็วกับเวลา (v-t Graph)

กราฟ v-t แสดงว่าวัตถุมีความเร็วเท่าใดในแต่ละเวลา สามารถอ่านได้ทั้ง ความเร่ง (จากความชัน) และ การกระจัด (จากพื้นที่ใต้กราฟ)

📐
ความชันของกราฟ = ความเร่ง

\(a_x = \dfrac{\Delta v_x}{\Delta t}\) (สำหรับความเร่งขณะหนึ่ง ใช้ความชันของเส้นสัมผัส)

▦
พื้นที่ใต้กราฟ = การกระจัด Δx

พื้นที่เหนือแกนเวลา → Δx เป็นบวก, พื้นที่ใต้แกนเวลา → Δx เป็นลบ

↔️
กราฟขนานแกนเวลา = ความเร็วคงตัว (a = 0)

\(v_x\) ไม่เปลี่ยนแปลง → ไม่มีความเร่ง

📉
กราฟเส้นตรงเอียง = ความเร่งคงตัว

ความชันคงที่ตลอด แสดงว่าความเร่งไม่เปลี่ยนแปลง

ตัวอย่าง 2.8 (หน้า 72–74, รูป 2.14): จากกราฟ v-t ที่ v(0 ถึง 1.0 s) = +4.0 m/s และ v(1.0 ถึง 3.0 s) = −8.0 m/s:
  • ช่วง t = 0 ถึง 1.0 s: พื้นที่ = (+4.0)(1.0) = +4.0 m = Δx ช่วงนั้น ✓
  • ช่วง t = 1.0 ถึง 2.0 s: พื้นที่ = (−8.0)(1.0) = −8.0 m = Δx ✓
  • ช่วง t = 1.0 ถึง 3.0 s: พื้นที่รวม = −8.0 + (−8.0) = −16.0 m ✓
  • ช่วง t = 0 ถึง 3.0 s: พื้นที่รวม = +4.0 + (−16.0) = −12.0 m ✓
พื้นที่ใต้กราฟ v-t = การกระจัด เสมอ
สรุปสิ่งที่อ่านได้จากกราฟ v-t
ลักษณะกราฟ ความหมาย
เส้นตรง slope > 0 ความเร่งคงตัวและเป็นบวก (เร็วขึ้น)
เส้นตรง slope < 0 ความเร่งคงตัวและเป็นลบ
เส้นขนานแกนเวลา ความเร็วคงตัว (a = 0)
v > 0 (กราฟอยู่เหนือแกน) เคลื่อนที่ทิศบวก
v < 0 (กราฟอยู่ใต้แกน) เคลื่อนที่ทิศลบ
กราฟตัดแกนเวลา (v = 0) วัตถุหยุดชั่วขณะ แล้วกลับทิศ

กราฟ v-t: ความชัน = ความเร่ง, พื้นที่ = การกระจัด

a-t

2.5.3 กราฟระหว่างความเร่งกับเวลา (a-t Graph)

กราฟ a-t แสดงความเร่งของวัตถุในแต่ละเวลา ข้อมูลที่ได้จากกราฟนี้คือ การเปลี่ยนความเร็ว (จากพื้นที่ใต้กราฟ)

▦
พื้นที่ใต้กราฟ a-t = การเปลี่ยนความเร็ว Δvx

\(\Delta v_x = a_x \cdot \Delta t\) (เมื่อ \(a_x\) คงตัว)

📏
เมื่อความเร่งคงตัว → กราฟเป็นเส้นขนานแกนเวลา

ค่า \(a_x\) ไม่เปลี่ยนตามเวลา

🔢
หาความเร็วที่เวลาใด ๆ จาก Δv

\(v_{fx} = v_{ix} + \Delta v_x\) = \(v_{ix}\) + พื้นที่ใต้กราฟ

ตัวอย่าง 2.9 (หน้า 75–77, รูป 2.15): วัตถุมีความเร่งคงตัว \(a_x = +8.0\) m/s² ตลอด และ\(v_{ix} = -8.4\) m/s
  • ช่วง t = 0 ถึง 0.8 s: Δv = (+8.0)(0.8) = +6.4 m/s
    \(v_x(0.8) = (-8.4) + 6.4 = \) −2.0 m/s
  • ช่วง t = 0 ถึง 2.3 s: Δv = (+8.0)(2.3) = +18.4 m/s
    \(v_x(2.3) = (-8.4) + 18.4 = \) +10.0 m/s
ตัวอย่าง 2.10 (หน้า 78–79): กราฟ a-t เป็นสามเหลี่ยม (a เพิ่มจาก 0 ถึง 64 m/s² ใน 16 s) พื้นที่ = ½ × (16.0)(64.0) = +512 m/s
ถ้า \(v_{ix} = +10.0\) m/s → \(v_{fx} = 10.0 + 512 = \) +522 m/s

กราฟ a-t: พื้นที่ใต้กราฟ = การเปลี่ยนความเร็ว Δv

  • ความชันเส้นสัมผัส ณ จุด = \(v_x\) ขณะหนึ่ง

  • \(v_{fx} = v_{ix} +\) พื้นที่

2.6 สมการสำหรับการเคลื่อนที่แนวตรง

เลือกกรณีการเคลื่อนที่แล้วใส่ค่าเพื่อคำนวณปริมาณที่เกี่ยวข้อง

สรุปสมการการเคลื่อนที่แนวตรง (ความเร่งคงตัว)

ใช้สัญลักษณ์ตามหนังสือเรียน สสวท. โดย \(u_x\) = ความเร็วต้น, \(v_x\) = ความเร็วปลาย, \(a_x\) = ความเร่งคงตัว, \(t\) = เวลา, \(\Delta x\) = การกระจัด สมการทั้ง 4 นี้ใช้ได้เฉพาะเมื่อ ความเร่งมีค่าคงตัว เท่านั้น

สมการที่ 1 — หา \(v_x\)
\[v_x = u_x + a_x t\]
ใช้เมื่อ: รู้ \(u_x, a_x, t\) → หา \(v_x\)
สมการที่ 2 — หา \(\Delta x\) จากความเร็วเฉลี่ย
\[\Delta x = \left(\frac{u_x + v_x}{2}\right)t\]
ใช้เมื่อ: รู้ \(u_x, v_x, t\) → หา \(\Delta x\)
สมการที่ 3 — หา \(\Delta x\) จากเวลา
\[\Delta x = u_x t + \frac{1}{2}a_x t^2\]
ใช้เมื่อ: รู้ \(u_x, a_x, t\) → หา \(\Delta x\)
สมการที่ 4 — ไม่มี \(t\)
\[v_x^2 = u_x^2 + 2a_x \Delta x\]
ใช้เมื่อ: รู้ \(u_x, a_x, \Delta x\) → หา \(v_x\) (หรือกลับกัน)

คำอธิบายตัวแปรทั้งหมด: \(u_x\) = ความเร็วต้นในแนวแกน x (m/s)  |  \(v_x\) = ความเร็วปลายในแนวแกน x (m/s)  |  \(a_x\) = ความเร่งคงตัวในแนวแกน x (m/s²)  |  \(t\) = เวลา (s)  |  \(\Delta x\) = การกระจัดในแนวแกน x (m)

ความเร็วคงตัว (\(a = 0\))

เมื่อ \(a = 0\) ความเร็วไม่เปลี่ยนแปลง และ \(\Delta x = v_x \cdot t\)

\[x = x_0 + v_x t\]

\(x\) = ตำแหน่ง ณ เวลา \(t\) (m), \(x_0\) = ตำแหน่งเริ่มต้น (m), \(v_x\) = ความเร็วคงตัว (m/s), \(t\) = เวลา (s)

x = 30.00 m

ความเร่งคงตัว

ใช้เมื่อความเร่งมีค่าคงตัวตลอดช่วงเวลา

\[v_x = u_x + a_x t\] \[\Delta x = u_x t + \tfrac{1}{2}a_x t^2\]

\(u_x\) = ความเร็วต้น (m/s), \(a_x\) = ความเร่ง (m/s²), \(t\) = เวลา (s), \(\Delta x\) = การกระจัด (m)

v = 8.00 m/s, Δx = 20.00 m

2.7 การตกแบบเสรี (Free Fall)

การตกแบบเสรี คือการเคลื่อนที่ภายใต้แรงโน้มถ่วงของโลกเท่านั้น โดยไม่คิดแรงต้านอากาศ วัตถุทุกชนิดจะมีความเร่งลงด้านล่างเท่ากัน ไม่ขึ้นอยู่กับมวล (ทดลองโดยกาลิเลโอ)

ความเร่งโน้มถ่วง
\[g \approx 9.8\,\text{m/s}^2\]
  • \(g\) = ความเร่งโน้มถ่วงของโลก ≈ 9.8 m/s² (ใช้ค่า 9.80 m/s² ในตำรา สสวท.)
  • ถ้ากำหนดให้ทิศลงเป็นบวก: \(a = +g = +9.8\,\text{m/s}^2\)
  • ถ้ากำหนดให้ทิศขึ้นเป็นบวก: \(a = -g = -9.8\,\text{m/s}^2\)
สมการการตกอิสระ (ปล่อยจากหยุดนิ่ง, กำหนดทิศลงเป็นบวก)
\[ y = \frac{1}{2}g t^2 \qquad v_y = g t \qquad v_y^2 = 2gy \]
  • \(y\) = ระยะตกในแนวดิ่ง (m)
  • \(v_y\) = ความเร็วในแนวดิ่ง (m/s)
  • \(g\) = ความเร่งโน้มถ่วง = 9.8 m/s²
  • \(t\) = เวลา (s)
กรณีทั่วไป (โยนขึ้นหรือโยนลงด้วยความเร็วต้น)
\[ \Delta y = u_y t + \frac{1}{2}(-g)t^2 \qquad v_y = u_y - gt \]
  • \(u_y\) = ความเร็วต้นในแนวดิ่ง (m/s) — เป็นบวกถ้าโยนขึ้น
  • \(v_y\) = ความเร็วปลาย (m/s)
  • ที่จุดสูงสุด: \(v_y = 0\) แต่ยังคงมีความเร่ง \(g\) ลงอยู่
ข้อสังเกต: มวลของวัตถุไม่มีผลต่อความเร่งในการตกเสรี วัตถุหนักและเบาตกถึงพื้นพร้อมกัน (ในสุญญากาศ)

ทดลองปล่อยวัตถุ

t = 2.0 s, ระยะตก 19.60 m, v = 19.60 m/s ลง
